项目架构

```
├── app					(api层)
│   ├── admin        	(web接口)
│   └── web         	(admin接口)
├── common        	   	(公共文件)
│   ├── dto        		(数据传输对象)
│   ├── global        	(全局对象)
│   ├── initialize      (初始化)
│   └── utils        	(工具包)
├── config            	(配置文件) 
├── constant            (常量/枚举) 
├── docs            	(swagger文档目录)
├── middleware          (中间件)                    
├── model      			(数据模型)                                                  
├── router      		(路由层) 
├──	go.mod          	(相关依赖)
├── go.sum          	(相关依赖)
└── main.go         	(入口文件)
```
